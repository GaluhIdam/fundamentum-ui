Technology Stack

Fundamentum UI adalah Design System yang digunakan sebagai panduan dalam pengembangan produk di project Quantum. Implementasinya sendiri menggunakan Angular Framework . Untuk menjadi contributor di project ini ada beberapa tools yang disyaratkan untuk diinstall terlebih dahulu, yaitu:

  • Node.js ^20.9.0, Node.js — Download (nodejs.org)
  • {{ "Angular CLI npm install -g @angular/cli" }}
  • Git Git - Downloads (git-scm.com)
  • Visual Studio Code Visual Studio Code - Code Editing. Redefined

Untuk tambahan bisa juga ditambahkan Visual Studio Code extension seperti:

  • Angular Language Service Angular Language Service - Visual Studio Marketplace
  • Prettier - Code formatter Prettier - Code formatter - Visual Studio Marketplace
  • Angular Snippets Angular Snippets (Version 16) - Visual Studio Marketplace
  • ESLint ESLint - Visual Studio Marketplace
