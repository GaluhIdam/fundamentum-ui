Horizontal Stack Component

The HorizontalStackComponent is a UI component designed to display progress bars with customizable segments. Each segment represents a percentage of progress and is colored based on the provided configuration. The component is built as a standalone Angular component and is designed for reuse across different parts of your application.

Component Structure

  • progress: An array of objects representing the percentage and color of each progress segment. This input is required for the component to render properly.
  • showPercentage: A string input that determines how the percentage label is displayed within the progress bar. It accepts the following values:
    • 'inline': Displays the percentage as an inline label within the progress segment.
    • 'tooltip': Shows the percentage in a tooltip when hovering over the progress segment.
    • 'none': Does not display the percentage label at all.
    By default, it is set to 'none'.
  • size: A string input that defines the size of the progress bar. Acceptable values are 'xs', 's', 'm', 'l', and 'xl', with 'm' being the default.

Key Features

  • Dynamic Progress Bars: The component takes an array of percentage and color pairs, rendering each segment of the bar with the specified width.
  • Customizable Colors: You can assign any color from a predefined set to each progress segment, making it visually distinct.
  • Optional Percentage Display: The percentage label inside each segment can be toggled on or off using the showPercentage input.

Example Use Case

This component can be used for representing progress in various contexts, such as task completion, project milestones, or health bars in games. It provides flexibility in visual representation while being simple to implement across different areas of the application.

SCSS Styling

The styles for this component can include classes like .hr-stack for the horizontal stack styling and color classes for each segment's background color. The width is dynamically adjusted based on the percentage value.
