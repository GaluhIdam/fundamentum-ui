Facet Button

FuiFacetButtons are to be used when allowing lists with multiple search params to be filtered down by these particular params. They allow for an icon node and/or quantity to be passed. You can also indicate the current selection with isSelected. Other props include isDisabled and isLoading (which will swap the quantity indicator with a loading icon).
