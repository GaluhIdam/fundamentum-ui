Advance Filter

The fui-advance-filter component is a customizable filter interface for advanced data searching and filtering in Angular applications. It allows users to define multiple filter sections, each containing input forms for targeted criteria. The component includes options for text input (fui-input-field), selection from predefined lists (fui-combo-box), and date ranges (fui-date-range), making it adaptable to various data types. The component's structure supports configurable labels, values, and placeholders, with event emitters like actionOut for handling user actions.
