Filter group

Filter buttons Use FuiFilterGroup to wrap FuiFilterButtons into a container that looks nice against form fields (like search). These buttons are used in two different patterns. The most simplest use is that of an on/off pattern to show whether a filter is on. Add the prop withNext to remove the border between it and the next FuiFilterButton to visually group similar or opposite style filters.
