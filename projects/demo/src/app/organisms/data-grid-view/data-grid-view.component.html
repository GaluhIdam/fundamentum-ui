Data Grid

FuiDataGrid is for displaying large amounts of tabular data. It is a better choice over FUI tables when there are many columns, the data in those columns is fairly uniform, and when schemas and sorting are important for comparison. Although it is similar to traditional spreedsheet software, FuiDataGrid's current strengths are in rendering rather than creating content.
