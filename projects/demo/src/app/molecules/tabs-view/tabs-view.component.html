Tab

Use tabs to organize related but unique content for a single idea or subject. Tabs show and hide content using in-page navigation UI. FuiTabs is a wrapping component that requires FuiTab components as direct children. You control the displayed contents and current state through props on FuiTab like isSelected and onClick. Use the prepend and append tab props to add content before and after the tab label respectively.
